# Screenshot Folder

Store submission screenshots in this folder using the following naming pattern:

- `01-homepage.png`
- `02-products-list.png`
- `03-product-detail.png`
- `04-cart.png`
- `05-checkout.png`
- `06-order-confirmation.png`
- `07-account.png`
- `08-orders.png`
- `09-admin-products.png`
- `10-admin-create-edit.png`

If you need more screenshots, keep the same numeric order and use short descriptive names.
